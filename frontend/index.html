V Room

This is a virtual conference room that replicates physical meeting rooms or classrooms. In this way, there is no need for guests and team members to be in the same geographical area to carry out a particular activity or task. They can connect from any device! Whether they are mobile devices, laptops, or desktop computers. The only requirement to be able to use the VRoom, is that the device with which you want to enter has an internet connection!
